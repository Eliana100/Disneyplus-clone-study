# Disney Plus UI Clone

Projeto de Front-end que recria a interface do Disney+ para fins educacionais.
O objetivo é praticar HTML, CSS e JavaScript, com foco em layout, organização e interações da interface.

⚠️ Este projeto é apenas para estudo e não possui qualquer vínculo com a Disney.

## O que foi praticado

- Estruturação de layout baseada em interface real
- Organização de arquivos e assets
- Manipulação de dados com JavaScript
- Interações e comportamento da interface
- Boas práticas de Front-end

## Inspiração e referências

Este projeto foi inspirado em um repositório base e em conteúdos educacionais.
Sendo o foco principal: consolidar conhecimentos de HTML, CSS e JavaScript em um projeto Front-end completo.

Conteúdo de apoio utilizado:
